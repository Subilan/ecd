# Internationalization

## 设计

`i18n/` 包提供中/英文界面字符串管理。

### 类型

```go
type Lang string
const (
    LangEN Lang = "en"
    LangZH Lang = "zh"
)
```

### 翻译表

使用 `map[string]stringEntry` 结构：

```go
var table = map[string]stringEntry{
    "key": {LangEN: "English text", LangZH: "中文文本"},
}
```

英文作为回退语言：当请求的语言找不到对应翻译时，返回英文字符串。

### API

```go
T("key", args...)          // 获取当前语言的翻译，支持 fmt.Sprintf 占位符
TWithLang(lang, "key", ...)  // 指定语言
SetLang(LangEN)            // 切换当前语言
GetLang() Lang             // 获取当前语言
```

### 占位符约定

| 占位符 | 用途 | 示例 |
|--------|------|------|
| `%s` | 单词、命令名 | `"synonym.not_found": "未找到 '%s' 的同义词。"` |
| `%d` | 数量 | `"synonym.found": "找到 %[2]s 的 %[1]d 个同义词"` |
| `%[1]d` / `%[2]s` | 显式参数索引（中英文语序不同时） | 同上 |

### 键名约定

- `source.<name>` — 词典名（`collins`/`oxford`）
- `label.<name>` — 字段标签（`definition`、`example`、`synonym`…）
- `note.<name>` — 笔记类型标签（`usage`、`quotation`、`phrase`…）
- `interactive.<name>` — 交互模式通用消息
- `search.<name>` — 搜索结果消息
- `add.<name>` / `del.<name>` — 添加/删除闪卡消息
- `review.<name>` — 复习流程消息
- `deck.<name>` — 牌组统计消息
- `common.<name>` — 跨页面通用消息
- `footer.<name>` — 底部栏消息
- `help.<name>` — 帮助页面消息

### 已知限制

- 搜索框 placeholder 在 `newSearchModel()` 时设定，语言切换后不会自动更新（需重启应用）
- CLI 模式的非 TTY 输出始终使用环境语言
